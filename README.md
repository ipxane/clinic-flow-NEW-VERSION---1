# Welcome to Clinic Pro

## Project info

Clinic Pro is a comprehensive clinic management system designed for professional healthcare providers.

## How can I edit this code?

If you want to work locally using your own IDE, you can clone this repo and push changes.

The only requirement is having Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd clinic-pro

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

You can deploy this project to any static hosting service like Vercel, Netlify, or GitHub Pages.

```sh
npm run build
```
